Long Live the Web

From the single browser and website, the web has seen itself row to a powerful tool where contributors who worked to expand it can access almost anything. While most people might interchange the terms web and internet, the World Wide Web is only but an application running on the internet. In contrast, the internet is a worldwide network of computers with various rules governing its use. As wide as the world wide web might sounds, more innovations are underway with prospects of introducing things such as free bandwidth, open data, social machines and web science are being introduced. As it stands, we already have open data, such as the United States data.gov website which allows individuals to access data posted on the platform on various subjects easily. Another active innovation is with the social machines seen from websites that allow people to rank sites and comment on various things a practice most popular in businesses such as hotels. With the principles of the web being preserved, there is so much the web has to offer in the future.
